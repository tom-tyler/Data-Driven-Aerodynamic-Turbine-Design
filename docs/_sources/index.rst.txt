Welcome to turbine_design's documentation!
==========================================

**turbine_design** is a turbine aerodynamic design tool in the form of a Python package. The package consists of two primary modules: :doc:`data_tools` for reading and manipulating CFD data, and the :doc:`turbine_design` design tool.

Check out the :doc:`usage` section for general information, including how to
:ref:`install <installation>` the project.

.. note::

   This project is under development
